About the project

Django Map Filter is simple project which shows how to enable nice and easy way for users to filter and query map objects based on their attributes.
Although all map objects here are related to model 'ApartmentsNY' it can easily be modifed and replaced with some any other model (depending on your needs).
Please have in mind that this is just basic version and that it could easily be extended depending on your needs.

Used Technologies

  • Backend
    • Python (Django)
  • Frontend
    • HTML5
    • Twitter Bootstrap
    • Javascript, JQuery
    • LeafletJS
  • Database
    • PostgreSQL (PostGIS)
